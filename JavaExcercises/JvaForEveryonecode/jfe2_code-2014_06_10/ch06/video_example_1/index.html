Removing Duplicates from an Array
Understand the Problem
  • Given: A sequence of items with repetitions
  • Example: Email addresses
  • Goal: Remove all duplicates
  • Don't change the order of the other items
Simulate by Hand
Design an Algorithm 
Remove all duplicates
start = 0
while start < currentSize
   remove a[start] from a[start+1]...a[currentSize - 1]
   Update currentSize
   start++

Remove from tail (a, currentSize, start, toBeRemoved)
source = start
target = start
removed = 0
while source < currentSize
   if a[source] equals toBeRemoved
     removed++
   else
     a[target] = a[source]
     target++
   source++
return removed
