Building a Book Index
Understand the Problem
  • Book index lists on which page each word occurs
  • Real book index lists “important” words only; we'll list all words
  • Example text: Alice in Wonderland from Project Gutenberg
  • Text has no page numbers; we'll index chapters instead
  • Generic terminology: Segment; may be page or chapter
Choose Data Structures
  • Index lists segment numbers for each word
  • Each word can occur in multiple segments
  • Map words to sets of segment numbers
  • Ex.: "whiskers" -> {1, 4, 11}
  • Map keys are strings
  • Ex.: "whiskers"
  • Map values are sorted sets of integers
  • Ex.: {1, 4, 11}
  • Keys should also be sorted
  • Choose a
    TreeMap<String,TreeSet<Integer>>
