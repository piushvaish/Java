Designing a Baby Naming Program
Understand the Problem
  • Read file with a large number of boy and girl names
  • User specifies the first letter
  • User chooses boy or girl names
  • Choices for length: short, medium, long, or any combination
  • All matching names are displayed whenever a selection changes
Sketch the User Interface
Required Components
  • A combo box for the first letter
  • Radio buttons for boy/girl
  • Check boxes for the length options
  • A scrolling text area for the result
Layout
  • A panel to hold the radio buttons
  • Another panel to hold the check boxes
  • Another panel to hold the combobox (so it won't grow)
