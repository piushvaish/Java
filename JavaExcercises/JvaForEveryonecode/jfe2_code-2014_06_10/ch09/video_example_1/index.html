Building a Discussion Board
Program Requirements
  • Students and instructors can add messages
  • Users can respond to messages
  • Response title = "Re: " + message title
  • When displaying a response, also display the original message
  • Instructors have additional privileges: purge messages, add users

User Interface

  • Menu 
    S)how messages
R)eply to message
N)ew message
L)ogout
P)urge message (instructor only)
A)dd user (instructor only)
  • Show command shows list of message titles
    • Prompts user for message number
    • Selected message is displayed
    • Display includes message author, title, text
Design Classes

Inheritance Hierarchy

Responsibilities of User class

  • Provide user-specific menu
  • Provide user name (displayed as message author)
  • Authenticate password
  • Instructor is a user, but with a different menu
public class User
{
   public User(String aName, String aPassword)
   public String getName()
   public boolean authenticate(String aPassword)
   public String getMenu()
}

public class Instructor extends User
{
   public String getMenu()
}

Responsibilities of Message class

  • Display message
  • Provide title name (displayed in message, message summary
  • Response is a message, but title is "Re: " + original message title
  • Response display contains original message
public class Message
{
   public Message(User author, String title, 
      String text)
   public String getTitle()
   public void display()
}

public class Response extends Message
{
   public Response(User author, Message original, 
      String text)
   public String getTitle()
   public void display()
}
