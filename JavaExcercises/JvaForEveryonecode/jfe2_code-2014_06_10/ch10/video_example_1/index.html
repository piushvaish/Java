Solving Crossword Puzzles
Understand the Problem
  • Read file with a large number of words
  • User specifies the known letters
  • Blanks indicate unknown letters
  • Click on button to search for all matches
  • Buttons to get more or fewer letters
Sketch the User Interface
Required Components
  • A text field for each letter
  • Buttons for adding, removing, and search
  • A text area for displaying the result
  • The text area should scroll
Listeners
  • Add button: add a text field
  • Remove button: remove a text field
  • Search button: put matches in text area
