Computing a Document's Readability
Understand the Problem
  • Count all sentences in the file. Count all words, and all syllables in the words.
  • Index = 206.835 - 84.6 x (Syllables/Words) - 1.05 x (Words/Sentences)
  • Gives an indication of the difficulty: 
    Comics                95
Consumer ads          82
Sports Illustrated    65
Time                  57
New York Times        39
Auto insurance policy 10
Internal Revenue Code  6

Algorithmic Questions

  • What is a word?
    • Any sequence of characters delimited by white space
  • What is a sentence?
    • A sentence is ended by a period, colon, semicolon, question mark, or exclamation mark.

Algorithmic Questions

  • How can we count syllables?
    • Syllable = group of adjacent vowels (a, e, i, o, u, y)
      real — 1 syllable, regal—2 syllables
    • An e at the end of a word doesn't count.
      rate—1 syllable
    • Each word has at least one syllable, even if the previous rules give a count of 0.
      the—1 syllable

User Interface Questions

  • How is the input file specified?
    • On the command line
    • If none is specified, prompt the user in the console window.
  • What is the output format? 
    Words: xxx
Syllables: xxx
Sentences: xxx
Index: xxx
Stepwise Refinement 
If no command line argument
  Prompt user for file name
Construct a scanner from the file
(If file doesn't exist, exit.)
Keep counters sentences, words, syllables
For each word in the input
  words++
  Count syllables in word and add to syllables counter (how?)
  If word ends in . : ; ? !
    sentences++
Compute and display index  
Count syllables in word
count = 0
InsideVowelGroup = false
For each character 
  If the character is a vowel
    If not insideVowelGroup
      count++
      insideVowelGroup = true
  Else
    insideVowelGroup = false
If word ends in e, count--
If count == 0, count = 1
